Deployment Overview

01

Clone & Install

Clone the repository and install dependencies. pip install -r requirements.txt handles all Python constraints.

02

Initialize Server

Run python server.py. This launches the local inference engine on localhost:5000.

03

Load Extension

Enable Developer Mode in Chrome and load the unpacked extension/ directory.
